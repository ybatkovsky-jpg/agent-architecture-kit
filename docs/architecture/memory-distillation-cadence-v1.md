# Memory Distillation Cadence v1

## Purpose

Distillation closes the gap between:
- understanding something important;
- and preserving it in a durable form that helps future execution.

## Distill-worthy categories

- decisions
- patterns
- anti-patterns
- blockers with reuse value
- durable references

## Not distilled by default

- raw chatter
- full transcripts
- every tiny step
- transient noise with no reuse value

## Triggers

Distillation should happen when:
- a decision is accepted;
- a bounded slice yields a reusable lesson;
- a blocker reveals a systemic boundary;
- a canonical artifact is created;
- a workstream closes and context would otherwise be lost.

## Cadence layers

### Immediate distillation
Right after a significant trigger.

### End-of-slice distillation
At the end of a bounded execution slice.

### Periodic consolidation
A later pass that turns repeated signals into more stable references.
