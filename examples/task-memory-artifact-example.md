# Task / Memory / Artifact example

User says: "Create a task for memory regression hardening."

Correct result:
- Task registry gets a real task record.
- Artifact layer may get a spec or implementation note.
- Memory may get a distilled rule if the decision has long-term reuse value.

Incorrect result:
- only an artifact file exists;
- no task record exists;
- later the system acts as if the task was formally created.
