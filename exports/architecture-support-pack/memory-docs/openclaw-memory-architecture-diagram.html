Today

Current Logic

Typed layer уже существует, но пока ещё не задаёт весь serving. Основной operational truth сейчас идёт через lexical corpus + heuristics + authority ranking, а typed Memory Core остаётся усилением, но не доминирующим слоем.
Goal

Desired Logic

Typed Memory Core становится first-class plane: он задаёт более структурированную и предсказуемую подачу памяти, а lexical retrieval остаётся сильным fallback и backing-evidence substrate, а не единственным рабочим мотором.
Target path: keep Postgres backbone → strengthen freshness → promote typed serving → split orchestration → add trace/debug → rollout by gates
